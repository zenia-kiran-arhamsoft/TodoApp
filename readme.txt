- Create a project folder the run npm init command and answer some questions related to your app, it will create a basic package.json file.
- Initiate a git repository by command git init
- Install dependencies
    - express
    - mongoose
    - nodemon 
    - cors
- Create .env file
- Create  a file named .gitignore at the root of your project and add
    node_modules
    .env
    package-lock.json

- Create index.js or app.js file which will serve as an entry point. Here index.js is created at /src/index.js
- Start project by either by 
    node ./src/index.js 
    or 
    npm run dev ( preferrable )
